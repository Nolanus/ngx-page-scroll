NgxPageScroll Demo application

This small demo application showcases the usage of the ngx-page-scroll library. Please refer to the Readme for documentation and instructions. Whenever you encounter an error or have a suggestion for enhancement, don't hesitate to file an issue at the project repo on GitHub.

For debugging purposes you may enable scroll position logging by setting the checkmark at the top right. This results in all changes to the scrollTop/scrollLeft property to be logged to the console.

These two buttons have a "routerLink" and "href" attribute set. In combination with the pageScroll directive this results in a press opening the new route and scrolling to the referred DOM element.
